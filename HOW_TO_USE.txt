Requirements:
- .Net framework 4.0+

To use the application you must:
1. Build the application. c:\PATH\TO\PROJECT>build.bat
2. Run the application. (Example: c:\PATH\TO\PROJECT>output\RouteCalculator.exe sample_data.txt

To generate test, analysis and coverage reports, you must:
1. Run the build project with: c:\PATH\TO\PROJECT>build_all.bat
